# PRODUCT.md

# Doshomik IELTS Product Specification

## Mission

Help learners achieve their IELTS target band faster through modern learning science and AI assistance.

---

# Target Users

## Primary

IELTS Candidates

Age:

16–35

Regions:

* Bangladesh
* South Asia
* Global

---

# Core Value Proposition

One platform for:

* Learning
* Practicing
* Evaluating
* Tracking

---

# MVP Features

## Authentication

* Register
* Login
* Profile

---

## Vocabulary

Features:

* Word Lists
* Meanings
* Examples
* Synonyms
* Difficulty Levels

---

## Flashcards

Modes:

* Review
* Learn
* Test

Inspired by Quizlet workflows.

---

## Quizzes

Question Types:

* MCQ
* True/False
* Matching
* Fill in the Blank

---

## Writing Evaluation

User submits:

* Task 1
* Task 2

AI returns:

* Feedback
* Improvement Suggestions
* Estimated Band Range

---

## Speaking Practice

User uploads audio.

AI provides:

* Fluency Feedback
* Vocabulary Suggestions
* Pronunciation Notes

---

## Dashboard

Displays:

* Streak
* Progress
* Study Time
* Weak Areas
